# FileShare
FileShare is a filesharing web-app running on Node.js

## Requirements

- Node.js with express and multer installed
  Install express and multer like this:
```
npm i express multer
```

## Installation

1. Clone this repo
```
git clone https://github.com/Johann-Schneider/FileShare.git
```
3. Open the folder /FileShare/
4. Type node /src/server.js
5. Open your browser and go to localhost:3000 or the IP from your server.
